# OOPinCsharp

Dette projekt er et simpelt C# konsolprogram (Hello World).

## Kom i gang

### Byg og kør

1. Åbn en terminal i projektmappen.
2. Kør følgende kommando for at bygge og køre programmet:

    dotnet run

### Krav
- .NET SDK (https://dotnet.microsoft.com/download)

## Projektstruktur
- OOPinCsharp.csproj – Projektfil
- Program.cs – Hovedprogram
